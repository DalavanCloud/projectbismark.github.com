Project BISmark
Open Development Nexus

Start here to access and participate in the development of Project BISmark's free & open source software and documentation. Learn more about the project at http://projectbismark.net.

Project management

  • Project wiki: project documentation and other development notes.
  • Project issue tracker: project bugs and feature requests that are not more appropriate for a specific repository.
  • Project issue dashboard: if you're a team member, this displays issues aggregated from all repositories in the project.

Software repositories

All of our repositories are hosted on Github at https://github.com/projectbismark/. Some of these repositories contain their own issue trackers and wikis.

Some noteworthy repositories include:

  • bismark-openwrt-overlay, the repository used to build the Openwrt router firmware image used by BISmark.
  • bismark-packages, the OpenWrt packages included in and used by the BISmark Openwrt firmware.
  • bismark-measurement-server, the tools that source and sink measurement traffic from BISmark routers on Measurement Lab and other measurement servers.
  • bismark-management-server, the tools that control the deployment of BISmark routers.
  • networkdashboard, the web app behind http://networkdashboard.org .
